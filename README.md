# This is A 2D Car Simulator
I built it using Pygame

## How to Run it :)
First thing first
Run 
 'pip install pygame'
After it is done installing either in your terminal or in Command Prompt
Run
'python main.py'  
Hit the Enter Key

## For Controls
**Driving**
* W/A/S/D  Or Arrow Keys - To steer, accelerate and brake/reverse ( it depends on the mode)
* Space Bar - Brake
* Q /E : Left and Right blinkers, use these before changing lanes to avoid penalties

Now for the **General & Menus**
* Enter: Start Game or Confirm
* Esc / P: Pause or Resume
* H: Honk
* D: To Open Driving School :(
